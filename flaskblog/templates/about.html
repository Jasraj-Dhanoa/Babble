General

This is a secure Blog Application that provides you with the ability to share your thoughts, ideas and experiences with the world!

Make a new account, Write your first blog and Simply POST.

API Service

This Application has a REST-API Service which allows users to interact with the application from various platforms.

Posts

Endpoint: http://127.0.0.1:5000/getPosts

Receive all posts made by current users on the blog application.

Specific Post

Endpoint: http://127.0.0.1:5000/getPost/#

Receive specific post from the blog application using post id.

Token Authetication

To access the endpoints below, you must use a timed token provided by our service. Please retrieve a token by sending a GET request to http://127.0.0.1:5000/api_login with username and password in body as JSON.

Add Post

Endpoint: http://127.0.0.1:5000/addPost

Add your post to our database by sending your user id, post title and post content. Reminder to include token in header of request.

Update Post

Endpoint: http://127.0.0.1:5000/updatePost

Update your post to our database by sending your user id, new post title and new post content. Reminder to include token in header of request.

Delete Post

Endpoint: http://127.0.0.1:5000/deletePost

Delete your post from our database by sending post id. Reminder to include token in header of request.
